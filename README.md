# WaizAI Platform

This repository now includes:

- **Backend API** (Express/Prisma) in the project root.
- **Frontend dashboard** (Vite + React) in `/frontend` inspired by `waizai-dashboard-v2.html`.

## Frontend quick start

```bash
cd frontend
cp .env.example .env
npm install
npm run dev   # start UI
npm run build # verify production build
```

Configure API/socket endpoints in `.env`.
